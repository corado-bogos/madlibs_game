# madlibs_game
This is a beginner-friendly Python Madlibs game where the user builds a short story by entering words.

The program:
• Prompts the user to enter adjectives, nouns, and a verb ending in “-ing”
• Stores user input in variables
• Builds a custom story using f-strings
• Displays the final generated story in the console

This project helps practice:
• User input handling
• String variables and concatenation/f-strings
• Basic program structure in Python
• Working with multiple inputs to create dynamic output
